Quick-Start Guide

  1. Download files from the Topheno application.
  2. Upload files to the Data Merger application.
  3. Select 2 uploaded files
  4. Press the Merge button
  5. Configure the Join settings
  6. Resolve all data conflicts
  7. Download the merged file

Less-Quick Guide

To produce a merged data file from two original source data files:

  1. Upload your original source files, i.e. the files containing the data that you want to merge. See data file in the glossary for the required format.
  2. Select two uploaded files that you want to merge.
  3. Configure the Join settings.
  4. Press the Merge button to produce a joined representation. See joined representation in the glossary for notes on importing and exporting these representations as files.
  5. If necessary, inspect the newly created joined representation and reconfigure the joined settings.
  6. Resolve all of the conflicts in the joined representation.
  7. Press the Flatten button.
  8. Download the merged data file. See merged file in the glossary for the exported format.

...

There are three parts to this Merger application:

  1. The Uploader - Upload your source data files. See data file in the glossary for the required format.
  2. The Joiner - Join your uploaded files together and inspect the joined representations.
  3. The Flattener - Flatten joined representations by resolving all their conflicts. You can then produce and download completely merged files, along with their provenance reports.
