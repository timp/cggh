MalariaGEN Calculated Data Merger

To produce a merged and flattened data file from two original source data files:

  1. Upload your original source files, i.e. the files containing the data that you want to merge.
  2. Select two uploaded files that you want to merge.
  3. Configure the settings for the merging process.
  4. Press the Merge button to produce a new merged file.
  5. If necessary, inspect the newly created merged file and reconfigure the merge settings.
  6. Resolve all of the conflicts in the merged file.
  7. Press the Flatten button.
  8. Download the merged and flattened data file.

There are three parts to this web application:

  1. The Uploader - Upload and manage your source files
  2. The Merger - Merge your source files and manage the merged files
  3. The Flattener - Flatten merged files by resolving their conflicts, then create and manage the flattened products

Glossary

conflict (or data conflict)
An inconsistent value given in two different uploaded files.
data file
A file containing data. The current version of the Merger application only supports text files with backslash-escaped double-quoted comma-delimited values with line-feed end-of-line characters.
flatten
The process of reducing two separate data files into one file by resolving all conflicts.
flattened file
A file that has been produce by merging two different...
merge
The process of combining two different data files.
merged file
A combined representation of the data from two different source files.
Merger repository
A secure file repository containing all of the uploaded data files, along with the merged and flattened data files.
Merger application
This web-based tool for producing flattened data files from uploaded data files.
uploaded file
A file that has been uploaded to the Merger repository containing the original data.
